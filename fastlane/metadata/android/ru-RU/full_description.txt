Внимание: Перед обновлением приложения обязательно сделайте резервную копию.

Современный менеджер паролей работает без хранения ваших данных на серверах — все пароли только в ваших руках, поэтому нет повода беспокоиться.

1. Полностью автономная работа
2. Не содержит рекламы
3. Открытый исходный код (доступен для изучения)
4. Резервное копирование и восстановление ваших учётных данных
5. Поддержка временных одноразовых паролей (TOTP)

https://github.com/yogeshpaliyal/KeyPass
